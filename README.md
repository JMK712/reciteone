# 📚 背诵助手

> 一个优雅的知识点记忆训练工具，支持填空、揭示、浏览三种模式

## ✨ 功能介绍

### 🎯 三种背诵模式

| 模式 | 描述 |
|------|------|
| **📝 填空模式** | 点击空白处揭示答案，考验记忆效果 |
| **👁 揭示模式** | 先思考后显示完整答案，循序渐进 |
| **📋 浏览模式** | 快速浏览所有知识点 |

### 🔄 学习辅助

- **🔀 随机模式** - 打乱题目顺序，避免记忆位置
- **✗ 错题复习** - 专门复习做错的题目
- **📊 进度追踪** - 实时显示学习进度
- **💾 进度保存** - 自动保存学习进度，支持导入导出

### ⌨️ 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `空格` | 显示答案 / 揭示空白 |
| `→` | 下一条 |
| `←` | 上一条 |
| `1` | 记住了 |
| `2` | 没记住 |

## 🚀 快速开始

### 1. 准备数据文件

创建 `.jsonl` 格式的文件，每行一个JSON对象：

```jsonl
{"id":1,"chapter":"遗传与进化","title":"分离定律","full":"在生物的体细胞中，控制同一性状的遗传因子成对存在，不相融合；在形成配子时，成对的遗传因子发生分离，分离后的遗传因子分别进入不同的配子中，随配子遗传给后代。","keys":["体细胞","成对存在","不相融合","形成配子","发生分离","不同的配子"],"checkpoint":false}
{"id":2,"chapter":"遗传与进化","title":"自由组合定律","full":"控制不同性状的遗传因子的分离和组合是互不干扰的；在形成配子时，决定同一性状的成对的遗传因子彼此分离，决定不同性状的遗传因子自由组合。","keys":["遗传因子","分离","互不干扰","形成配子","自由组合"],"checkpoint":true}
```

### 2. 上传文件

打开 `背诵助手.html`，将 `.jsonl` 文件拖拽到上传区域或点击选择文件。

### 3. 开始背诵

选择适合你的背诵模式，开始学习！

## 📖 数据格式说明

### JSONL 字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | number | ✅ | 序号，从1开始 |
| `chapter` | string | ✅ | 章节名称 |
| `title` | string | ✅ | 4-10字标题 |
| `full` | string | ✅ | 知识点原文 |
| `keys` | array | ✅ | 挖空关键词（2-7个） |
| `checkpoint` | boolean | ❌ | 是否为复习节点 |

### keys 选取规则

**选**：定义中的核心术语、区分性描述词、数字/限定词、关键动词短语

**不选**：连接词（"因此"/"即"）、超过15字的长句、原文未出现的词、含义重复的词

### checkpoint 规则

每5-8条设置一个 `true`，放在小主题结束处，全文占比15%-20%
